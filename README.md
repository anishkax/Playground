# Playground

A playground repository for various experiments and projects.

## Projects

### [`finance-dashboard/`](./finance-dashboard)

A beautiful, modern finance dashboard frontend built with **React + Vite + TypeScript + Tailwind CSS**.

**Features:** Sidebar navigation, KPI cards with sparklines, cashflow panel, transaction history, budget tracker, virtual card summary, account health panel.

```bash
cd finance-dashboard
npm install
npm run dev   # → http://localhost:5173
```

See [`finance-dashboard/README.md`](./finance-dashboard/README.md) for full documentation.
